## 目标

将 reference 实验**产品化**为独立工具链 `bfpe`，满足 R1–R4（I/O 流、一文件一函数、命令行、合规 PE）。详见可行性报告 §1。

## 依赖与约束

- Windows 10/11 x64，Visual Studio 2022（ml64、cl、link）
- Python 3.9+（CLI / codegen）
- 构建策略：**完整链接生成 PE**，不对第三方 PE 做 patch
- 合规：继承 reference 五项约束（C1–C5），每次 `build` 末尾强制 `verify_pe.ps1`
